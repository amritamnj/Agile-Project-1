The Evolution of the Hyperlink: A Timeline

This interactive timeline showcases the history of hyperlinks, from their early conception to the web we know today. Slide left and Click on each event to learn more about them!
  1. 1945
    Vannevar Bush publishes an essay

    Vannevar Bush publishes the essay "As We May Think" describing the Memex, a microfilm-based machine for linking information.

  2. 1963
    Ted Nelson coins the term "link"

    Ted Nelson coins the term "link" while working on Project Xanadu, inspired by Bush's Memex concept. Xanadu aims to create a worldwide computer network with hyperlinks.

  3. 1968
    Expansion of Hyperlinks

    Engelbart's team expands hyperlinks to connect paragraphs between separate documents in NLS.

  4. 1987
    HyperCard for Apple Macintosh

    HyperCard allows hyperlinking within documents and between documents/applications.

  5. 1989
    The World Wide Web

    Tim Berners-Lee invents the World Wide Web, a global system of interconnected hypertext documents.

  6. 1991
    The Gopher Protocol

    The Gopher protocol becomes the first widely used open protocol for hyperlinking between internet sites.

  7. 1992
    Release of Lynx Web Browser

    The Lynx web browser is released, allowing users to navigate the World Wide Web and access hyperlinks across the internet.

  8. 1993
    Release of Mosaic Web Browser

    Mosaic web browser handles both Gopher and HTML links. HTML's ability to combine graphics, text, and hyperlinks surpasses Gopher's text-based menu structure.

  9. 1995
    HTML Standardization

    The World Wide Web Consortium (W3C) establishes HTML (HyperText Markup Language) as the standard language for creating web pages, including defining hyperlink syntax.

  10. Mid 1990s-Present
    Search Engines & Dynamic Content

    Search engines like AltaVista helped navigate the web's explosion. Later, technologies like AJAX enabled dynamic content, further expanding hyperlinks' potential. Today, hyperlinks remain the web's cornerstone, evolving with new trends.

As We May Think

Famous American engineer Vannevar Bush wrote a groundbreaking article titled "As We May Think" in 1945. This essay proposed the Memex, a ground-breaking gadget. Memex was created to store and retrieve data from microfilm. However, what really set it apart was its capacity to associate information, much like hyperlinks do on the internet today. Even though microfilm technology was restricted, the Memex paved the way for future information management systems.

Doug Engelbart's Mother of All Demos

In a groundbreaking 1968 demonstration, Douglas Engelbart introduced a system called NLS that featured two innovations ahead of their time: the world's first computer mouse for navigating on-screen, and a precursor to today's hyperlinks, allowing users to jump between connected points within the system.

Gopher
Did you know?

In 1991, just a year after the Gopher Protocol was created, it accounted for 95% of the traffic on the internet. Its simplicity made it more accessible than the complex WWW system that would soon eclipse it.

Lynx Web Browser

Lynx is a customizable text-based web browser for use on cursor-addressable character cell terminals. As of 2024, it is the oldest web browser still being maintained, having started in 1992. Text-only surfing offers speed advantages that are especially noticeable on low-bandwidth internet connections. In Lynx, you can browse by either using the cursor keys to highlight the selected link or by having all the links on a page numbered and inputting the link's number. Many HTML features and SSL are supported in the current versions. While frames are identifiable by name and can be viewed as individual pages, tables are constructed using spaces. Although Lynx can't display all non-textual stuff on the web, like video and images, it can launch other applications that do, such an image viewer or a video player. A drawback of Lynx is that it does not support JavaScript, which many websites need to function properly, in contrast to the majority of web browsers.

Mosaic Web Browser

Mosaic was the first web browser to feature a graphical user interface that allowed images to be displayed inline with text on web pages, rather than in separate windows. Its release in 1993 by the National Center for Supercomputing Applications (NCSA) marked a significant leap forward in web technology and usability, paving the way for the browsers that followed and the explosive growth of the internet in the mid-1990s.

HTML Standardization

The World Wide Web Consortium (W3C), a global organization that works to define web standards, is in charge of managing HTML standardization. The goal is to secure the web's continuous expansion. The open web as it exists now would be far more chaotic and fragmented without the standardization efforts of groups like the W3C.

Key benefits:

  • Consistency Across Web Browsers: It guarantees that websites function consistently on various devices and browsers. Developers can save time and resources by not having to produce separate versions of the same website for different browsers.
  • Accessibility: Web material is more useable for individuals with disabilities when it adheres to accessibility criteria, which is supported by standard HTML. Inclusion is necessary to have a fair and accessible internet.
  • Maintenance Ease: Updating and maintaining a uniform codebase is simpler. Consistent code methods allow developers to optimize and fix web pages more quickly.
Search Engines & Dynamic Content

The mid-1990s witnessed two key developments that transformed how we navigate the web:

  • Search Engines: Search giants like AltaVista and Yahoo! emerged, making it easier to discover the vast amount of information connected by hyperlinks. This increased user engagement.
  • Dynamic Content and AJAX: By the late 1990s and early 2000s, technologies like server-side scripting and AJAX allowed webpages to update content without full page reloads. Clicking a hyperlink could now trigger dynamic changes, making the web more interactive and engaging.

Today and Beyond:

Hyperlinks remain the cornerstone of web navigation in 2024. As new technologies like single-page applications and touch-based interfaces emerge, hyperlinks continue to evolve, ensuring their place in shaping the future of the web.
